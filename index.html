Twitter Clone

I cloned Twitter with features such as logging in, signing up, posting tweets & comments, settings options that lets you change theme and a logout feature using a mix of tech tools: MongoDB, Express.js, React, Node.js and tailwind for styling. MongoDB stored user data and tweets, Express.js handled the backend API, React made the website interactive and For managing data, I used Redux Toolkit and the context API. I hosted the web server on Render.com and the front-end on Netlify.
